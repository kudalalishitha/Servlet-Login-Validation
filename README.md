First Servlet Login Project

About:
This is a simple Java Servlet project for Login.

Features:

Create first servlet

Login with predefined username and password
Name validation
Name starts with Capital letter
Name has minimum 3 characters
Password validation
Minimum 8 characters
At least 1 uppercase letter
At least 1 number
Exactly 1 special character

Tools Used:
Java
Servlet
Tomcat Server
